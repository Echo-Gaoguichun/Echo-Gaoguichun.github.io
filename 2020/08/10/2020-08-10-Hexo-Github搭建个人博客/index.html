三、部署Github

　　接下来就是把搭建好的个人博客发布到 Github 上。
　　首先进入 Github 网站，登录自己的账号，Create a new repository，Repository name 必须是 your github name + github.io （如果没有 Github 账号，网上资料都很全，可以搜索一下如何注册，绑定 Git，设置 ssh 密钥等，例如 ： Git 菜鸟教程 )

　　回到本地文件，在 blog 目录下有一个 _config.yml 配置文件，用 Edit Plus、Subline、NotePad++ 等编辑器打开，修改配置信息，特别要注意的是，在配置文件中，每个参数的：后面都需要加一个空格。
